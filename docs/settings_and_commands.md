# Server settings

+ `xor_telegram_link` Link to your Telegram channel
+ `xor_telegram_link` Link to your Discord server


# Rcon commands

+ `broadcastid` Broadcast message to player with client ID


# Chat commands

+ `/telegram` Shows the link to the Telegram channel you entered in `xor_telegram_link`
+ `/discord` Shows the link to the Discord server you entered in `xor_discord_link`
+ `/telehere (id or nickname)` Teleports a player to you. Only works if you are in rcon
+ `/deeppl (id or nickname)` Makes the player DeepFreeze. Only works if you are in rcon
+ `/undeeppl (id or nickname)` Makes the player UnDeepFreeze. Only works if you are in rcon
