# XOR



Based on DDNet: flavor of DDRace, a Teeworlds mod. See the [website](http://ddnet.tw) for more information.

Download DDNet++ version 0.0.7 for macOS/linux/windows [here on github](https://github.com/DDNetPP/DDNetPP/releases/tag/v.0.0.7).


## Cloning

Same as on [DDNet](https://github.com/ddnet/ddnet#cloning)


## Building

XOR is built in the same way as [DDNet++](./READMEDDNETPP.md#building) or [DDNet](https://github.com/ddnet/ddnet#building-on-linux-and-macos)


## Features

You can see all the functions and settings [here](./docs/settings_and_commands.md)

All functions and [settings from ddnet++](./docs/settings_and_commands_ddpp.md) are also present
